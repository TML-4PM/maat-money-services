# Maat Money Services

Cloud workers (Lambda/Supabase) implementing money & tax MCP tools.

## Structure

- `services/bank/` - Bank sync and CDR integration
- `services/ledger/` - Double-entry accounting engine
- `services/au_tax/` - BAS calculation and lodgement
- `services/rules/` - Transaction categorization rules engine
